# 💬 IA WhatsApp HUB

Monitoramento inteligente de grupos e mensagens do WhatsApp com IA.

## Funcionalidades
- 📊 Dashboard em tempo real
- 🔔 Detecção de menções a Marcos Sousa
- 🤖 Análise IA automática (Claude)
- 🎯 Monitoramento por palavras-chave (IA, CRM, follow-up, etc)
- ⚠️ Classificação de prioridades
- 👥 Visualização de grupos e mensagens

## Deploy
Conectado ao Vercel via GitHub. Deploy automático a cada push.

## Stack
- Frontend: HTML/CSS/JS puro
- Backend: Vercel Serverless Functions (Node.js)
- API: UazAPI (WhatsApp)
- IA: Claude (Anthropic)
